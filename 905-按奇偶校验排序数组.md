给定一个非负整数数组 A，返回一个由 A 的所有偶数元素组成的数组，后面跟 A 的所有奇数元素。

你可以返回满足此条件的任何数组作为答案。

 

示例：

输入：[3,1,2,4]
输出：[2,4,3,1]
输出 [4,2,3,1]，[2,4,1,3] 和 [4,2,1,3] 也会被接受。
 

提示：

1 <= A.length <= 5000
0 <= A[i] <= 5000  

```cpp
class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& A) {
        int size = A.size(), start=0, end=size-1;
        vector<int> result(size, 0);
        for(int i : A)
            if(i%2 == 0)
                result[start++] = i;
            else
                result[end--] = i;
        return result;
    }
};
```
